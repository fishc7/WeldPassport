# WeldPassport — Навигация

Единая точка входа по всем материалам проекта. Каждый документ доступен отсюда.
Код — в `09_Разработка/`. Данные с ПДн и временные файлы (`Работники/`, `tools/`,
`_Входящее/`) в навигацию не входят и в git не попадают.

> Ссылки на заметки — в формате Obsidian `[[путь/Имя|Название]]` (полный путь от
> корня хранилища), чтобы граф связей был однозначным. Ассеты (PDF/PNG/SVG) и папки —
> обычными Markdown-ссылками. Для чистого графа исключите из хранилища `.worktrees/`
> и `_archive/` (Settings → Files & Links → Excluded files) — там лежат копии-дубли.

## Как поддерживать навигацию

Новый документ → добавить строку в нужный раздел этого файла (и, если относится к
проектированию, в [[10_Проектирование_WeldPassport/00_Навигация_проектирования|навигацию по проектированию]]).
Один документ — одна основная ссылка; в других местах ставить ссылку, а не копию.
Ссылки на заметки — в формате `[[путь/Имя|Название]]`.
