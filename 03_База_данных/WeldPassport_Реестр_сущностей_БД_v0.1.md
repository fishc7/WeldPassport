# WeldPassport — Реестр сущностей БД v0.1

Источник: [Google Sheets](https://docs.google.com/spreadsheets/d/1LU-P_8gLrsrb7A3Y_VvyqkK4Y9vAR_Y1su-4ryuZn_c)

| № | Сущность | Назначение | Основные поля | Связи | Статус | Комментарий |
|---:|---|---|---|---|---|---|
| 1 | Проекты | Учет проектов сварочного производства | ID, название, заказчик, подрядчик, даты, статус | Объекты, изометрии, стыки | Черновик | Ядро системы |
| 2 | Объекты | Площадки/объекты внутри проекта | ID, проект, название, участок, статус | Проект, изометрии, стыки | Черновик | |
| 3 | Изометрии | Рабочие документы трубопроводов | ID, номер, проект, объект, ревизия, статус | Стыки, детали | Черновик | Нужна нумерация |
| 4 | Стыки | Единица учета сварочного соединения | ID, номер, изометрия, тип, диаметр, толщина, статус | Изометрия, факт сварки, НК, материалы | Черновик | Ключевая сущность |
| 5 | Работники | Общий список работников | ID, ФИО, табельный номер, должность, организация, статус | Сварщики, документы | Черновик | Не каждый работник является сварщиком |
| 6 | Сварщики | Работники, выполняющие сварку | ID, работник, категория, статус, клеймо | Работник, документы, допуски, факты сварки | Черновик | Учитывать обученных и аттестованных |
| 7 | Документы сварщика | НАКС, свидетельства, дипломы, обучение | ID, сварщик, тип документа, номер, дата, срок, файл | Сварщик, допуски | Черновик | Нужна проверка срока действия |
| 8 | Внутренние допуски | Разрешение организации на сварку | ID, сварщик, область, способ, дата, срок, статус | Сварщик, стык, факт сварки | Черновик | До допуска заказчика |
| 9 | Допуски заказчика | Согласование сварщика заказчиком | ID, сварщик, проект, заказчик, дата, срок, статус | Сварщик, проект, факт сварки | Черновик | |
| 10 | Факт сварки | Фиксация фактического исполнителя | ID, стык, сварщик, дата, смена, способ, мастер | Стык, сварщик, мастер, НК | Черновик | Главная проблема проекта |
| 11 | Участники подготовки стыка | Звеньевой/помощник/подготовка стыка | ID, стык, работник, роль, дата, комментарий | Стык, работник | Черновик | Отдельно от сварщика |
| 12 | НК | Результаты неразрушающего контроля | ID, стык, вид контроля, дата, результат, заключение | Стык, дефекты, ремонт | Черновик | |
| 13 | Сварочные материалы | Материалы, партии, сертификаты | ID, наименование, марка, партия, сертификат, срок | Выдача, расход, стыки | Черновик | Отдельная боль проекта |
| 14 | Выдача материалов | Учет передачи материалов в работу | ID, материал, кому выдано, дата, количество, объект | Материал, работник, проект | Черновик | |
| 15 | Исполнительные журналы | Формирование отчетности по сварке | ID, проект, период, тип журнала, статус | Стыки, факт сварки, НК | Черновик | Выходной документ |
