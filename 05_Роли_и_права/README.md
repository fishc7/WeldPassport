## Правило размещения

ТЗ по модулю роли хранится в папке этой роли (`<роль>/ТЗ/`). При создании нового ТЗ
добавлять на него ссылку в таблицу выше и, если материал затрагивает проектирование,
в `../10_Проектирование_WeldPassport/00_Навигация_проектирования.md`.

Исходные данные с ПДн (табели, выгрузки) внутри папок ролей в git не попадают
(см. `.gitignore`).
