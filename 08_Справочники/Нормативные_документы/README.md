# Нормативные документы по аттестации сварщиков

Официальные нормативы — основа справочников и правил модуля ОГС (аттестации, допуски,
способы сварки, положения, группы материалов и технических устройств). Эти же справочные
значения использует выгрузка `naks_data`.

## Документы

| Документ | Файл | Что задаёт |
|---|---|---|
| ПБ-03-273-99 | [ПБ-03-273-99.doc](<ПБ-03-273-99.doc>) | Правила аттестации сварщиков и специалистов сварочного производства: уровни, область, порядок |
| РД 03-495-02 | [РД 03-495-02.pdf](<РД 03-495-02.pdf>) | Технологический регламент проведения аттестации: коды способов сварки, положений, групп материалов, видов деталей и швов |
| РД 03-615-03 | [РД 03-615–03.pdf](<РД 03-615–03.pdf>) | Порядок применения сварочных технологий на опасных производственных объектах (аттестация технологий сварки) |

## Связь со справочниками WeldPassport

Из этих нормативов формируются справочники, на которые ссылаются аттестации сварщиков:

- способы (виды) сварки — например РАД, РД, МП, АФ;
- положения при сварке — Н1, В1, Г, П1, П2 и т.д.;
- группы свариваемых материалов — М01, М03, М11 и т.д.;
- группы технических устройств опасных производственных объектов;
- виды деталей, типы швов, виды соединений.

См. также: [ТЗ модуля ОГС](../../05_Роли_и_права/05_Главный_сварщик_ОГС/ТЗ/WeldPassport_ОГС_ТЗ_v0.1.md),
[Словарь терминов](../WeldPassport_Словарь_терминов_v0.1.md).
