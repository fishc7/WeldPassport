# Справочники сварки

Справочники для аттестаций сварщиков, наполнены из норматива
[РД 03-495-02](../Нормативные_документы/РД%2003-495-02.pdf) (Технологический регламент
проведения аттестации). Значения сверены с данными `naks_data` (совпадают).

Файлы для импорта (CSV, UTF-8, разделитель «,»):

- [Способы_сварки.csv](<Способы_сварки.csv>)
- [Положения_сварки.csv](<Положения_сварки.csv>)
- [Группы_материалов.csv](<Группы_материалов.csv>)

В аттестациях НАКС эти коды встречаются как многозначные поля (несколько значений через
`; `, например положение `В1; Г; Н1; Н2`; материал `М01; М11`). При импорте разбивать
по разделителю и сопоставлять со справочником.

## Способы сварки

Код, наименование, код по ISO 4063 (если есть), категория. Полный список — в CSV.
Основные:

| Код | Наименование | ISO 4063 |
|---|---|---|
| РД | Ручная дуговая сварка покрытыми электродами | 111 |
| РАД | Ручная аргонодуговая сварка неплавящимся электродом | 141 |
| МП | Механизированная сварка плавящимся электродом в активных газах | 135 |
| АФ | Автоматическая сварка под флюсом | 12 |
| П | Плазменная сварка | 15 |
| Г | Газовая сварка | 311 |

Всего в CSV: 33 кода (металлы, наплавка, контактная, пайка, полимеры).

## Положения сварки

| Код | ISO 6947 | Наименование |
|---|---|---|
| Н1 | PA | Нижнее стыковое и в «лодочку» |
| Н2 | PB | Нижнее тавровое |
| Г | PC | Горизонтальное |
| П1 | PE | Потолочное стыковое |
| П2 | PD | Потолочное тавровое |
| В1 | PF | Вертикальное снизу вверх |
| В2 | PG | Вертикальное сверху вниз |
| Н45 | H-L045 | Наклонное под углом 45 градусов |

## Группы свариваемых материалов

Полный список (М00–М64) — в CSV. Основные:

| Код | W-код | Наименование (кратко) |
|---|---|---|
| М01 | W01 | Углеродистые и низколегированные стали (σт до 360 МПа) |
| М03 | W03 | Низколегированные стали перлитного класса (σт свыше 360 МПа) |
| М11 | W11 | Высоколегированные аустенитно-ферритные и аустенитные стали |
| М21 | W21 | Чистый алюминий и алюминиево-марганцевые сплавы |
| М41 | — | Титан и титановые сплавы |
| М61 | — | Полиэтилен (PE) |
| М00 | — | Материалы вне обозначенных групп |

## Ещё не наполнено (из других нормативов)

- **Группы технических устройств опасных производственных объектов** (НГДО, ПТО, КО,
  ГО, ОХНВП, КТ и т.д.) — источник: ПБ-03-273-99 / РД 03-615-03; в `naks_data` есть
  колонка «Группы технических устройств».
- **Степень автоматизации** (для полимеров: СР, ССА, СВА, ЗНР, ЗНШ, ЗНА…) — РД 03-495-02.
- Виды деталей, типы швов, виды соединений.
