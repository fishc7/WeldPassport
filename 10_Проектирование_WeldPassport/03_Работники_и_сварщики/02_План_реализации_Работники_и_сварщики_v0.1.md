# «Работники и сварщики» v0.1 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Реализовать в PostgreSQL и SQLAlchemy согласованное ядро учета людей, трудоустройств, статуса сварщика, клейм, документов, аттестаций, допусков, дублей и проверки возможности назначения.

**Architecture:** Изменения выполняются поэтапно и преимущественно добавлением новых таблиц. Существующие таблицы `РАБОТНИКИ` и `СВАРЩИКИ` временно сохраняются как совместимый слой для уже созданных внешних ключей производственного модуля. Чистые правила нормализации, поиска дублей и проверки допуска размещаются отдельно от ORM, чтобы тестировать их без PostgreSQL.

**Tech Stack:** Python 3.11+, SQLAlchemy 2.x, PostgreSQL, psycopg 3, Alembic, pytest.

## 2. Стратегия совместимости

Новые таблицы становятся источником истины для людей и квалификаций. Существующие записи сохраняются:

- `РАБОТНИКИ` получает обязательную ссылку `ID_Человека` и продолжает обозначать совместимую запись трудоустройства для старых внешних ключей;
- `СВАРЩИКИ` получает ссылку на человека и остается совместимым техническим профилем для старых связей;
- новые периоды статуса сварщика хранятся отдельно;
- старые поля ФИО, организация, должность и клеймо не удаляются до перевода производственного модуля и импорта;
- миграция сначала заполняет новые таблицы, затем добавляет ограничения;
- downgrade удаляет только новые объекты и новые ссылки, не удаляя исходные строки старых таблиц.

- [ ] **Step 3: Реализовать минимальный нормализатор**

Определить неизменяемый `NormalizedName` и функцию:

```python
@dataclass(frozen=True)
class NormalizedName:
    source: str
    normalized: str
    surname: str | None
    first_name: str | None
    patronymic: str | None
    is_incomplete: bool


def normalize_name(value: str) -> NormalizedName:
    source = value.strip()
    normalized = source.casefold().replace("ё", "е")
    normalized = re.sub(r"[.,;]+", " ", normalized)
    normalized = re.sub(r"\s+", " ", normalized).strip()
    if not normalized:
        raise ValueError("Имя человека не может быть пустым")
    parts = normalized.split()
    surname = parts[0] if parts else None
    first_name = parts[1] if len(parts) > 1 else None
    patronymic = parts[2] if len(parts) > 2 else None
    is_incomplete = len(parts) < 3 or any(len(part) == 1 for part in parts[1:3])
    return NormalizedName(
        source=source,
        normalized=normalized,
        surname=surname,
        first_name=first_name,
        patronymic=patronymic,
        is_incomplete=is_incomplete,
    )
```
