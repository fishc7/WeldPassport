# Цель проекта

Создать единую цифровую платформу управления сварочным производством, объединяющую:

* управление персоналом;
* управление сварочными работами;
* контроль качества;
* исполнительную документацию;
* нормирование;
* аналитику;
* интеграции с корпоративными системами.

---

# Приоритеты

Приоритет обозначается следующим образом:

* 🔴 Critical — без этого система не может работать
* 🟠 High — необходимо для MVP
* 🟡 Medium — важно после MVP
* 🟢 Low — можно реализовать позже

# Definition of Ready

Задача может быть взята в работу, если:

* описана цель;
* определены входные данные;
* понятен ожидаемый результат;
* отсутствуют архитектурные противоречия.

---

# Definition of Done

Задача считается завершённой, если:

* реализован код;
* выполнены проверки;
* обновлена документация;
* изменения закоммичены;
* решение соответствует `PROJECT_RULES.md`.
