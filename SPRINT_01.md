# Sprint 01 — Foundation (Основа платформы)

**Sprint:** 01
**Статус:** Active
**Дата начала:** 2026-07-01

---

# Цель спринта

Создать работоспособную основу системы WeldPassport, на которой будут строиться все последующие модули.

К завершению спринта должна быть готова первая рабочая версия платформы с базовой архитектурой, подключенной базой данных, авторизацией и модулем управления персоналом.

---

# Основные результаты

К окончанию Sprint 01 должны работать:

* FastAPI Backend
* PostgreSQL
* Alembic
* React + Vite Frontend
* JWT авторизация
* Пользователи
* Роли
* Подразделения
* Работники
* Swagger API
