# Sprint 00 — Project Foundation

**Статус:** Active
**Дата начала:** 2026-07-01
**Цель:** подготовить WeldPassport к системной долгосрочной разработке.

---

## Цель спринта

Создать чистую, понятную и управляемую основу проекта до начала активной разработки функциональности.

Sprint 00 не про бизнес-функции, а про фундамент:

* структура репозитория;
* документация;
* архитектура;
* правила разработки;
* GitHub-процесс;
* чистый frontend/backend;
* подготовка к Sprint 01.
