### Sprint 01 — Foundation (начат 01.07.2026)

Цель: рабочая основа платформы — backend, PostgreSQL, Alembic, авторизация,
модуль персонала.

Ключевые эпики и статус (на 01.07.2026, всё «не начато» на момент переноса):

- **Платформа** — настройка backend, подключение PostgreSQL, первая
  Alembic-миграция, единая структура каталогов.
- **Пользователи и безопасность** — модель `User`, JWT-авторизация, роли
  (Администратор, ОК, ОГС, ПТО, ОТК, Производство, Руководитель).
- **Организационная структура** — CRUD организаций, подразделений, должностей.
- **Управление персоналом** — карточка работника, статус сварщика, клеймо,
  допуски, аттестации.
- **REST API** — Users/Workers/Departments API, Swagger/OpenAPI.

Критерии завершения спринта: backend и frontend запускаются, есть подключение
к PostgreSQL, работает авторизация, реализованы пользователи/роли/подразделения/
работники, доступна документация Swagger.

Definition of Done — см. `AGENTS.md` §11.
